# Interactive System with Vite

This project is an interactive application developed with modern JavaScript and packaged with **Vite** for optimal performance.

---

## Prerequisites

Before you begin, ensure you have the following installed:
* **Node.js**: Version 20.19.0 or higher (LTS recommended).
* **NPM**: Version 10.x or higher.

> [!TIP]
> You can verify your versions with `node -v` and `npm -v`.

---

## Installation and Configuration

Follow these steps to clone and run the project locally:

### 1. Clone the repository
Copy the GitHub link and paste it into your terminal:

```bash
git clone [https://github.com/Juanjosegiraldo/Module3_Week_1_2_3_4_Riwi.git](https://github.com/Juanjosegiraldo/Module3_Week_1_2_3_4_Riwi.git)
```

Note: The repository has four folders. Make sure you are in the user_story/week1 folder.

2. Access the project
Open the folder you just created:

```Bash
cd vite-project
```

3. Install dependencies
Download all necessary Node packages:

```Bash
npm install
```

4. Run the development server
Start the real-time work environment:

```Bash
npm run dev
```

Once executed, open your browser at: http://localhost:5173

Main Structure:

*index.html: Main entry point.

*src/interactive_system.js: Contains all system logic.

*public/: Static files (images, icons).

*.gitignore: Files excluded from Git (such as node_modules).
