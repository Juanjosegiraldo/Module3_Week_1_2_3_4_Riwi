# Data Management Project | Objects, Sets & Maps

This repository contains a JavaScript implementation for managing product data structures. Built with **Vite** for a fast and modern development experience.

---

## Task Overview

The project is structured into five core modules:

* **Task 1: **Product Constructor** - Using functions to instantiate product objects.
* **Task 2: **Set Logic** - Filtering duplicates and managing unique numerical data.
* **Task 3: **Map Integration** - Establishing relationships between categories and products.
* **Task 4: **Iteration Protocols** - implementing `for...in`, `for...of`, and `.forEach()`.
* **Task 5: **Data Validation** - Ensuring data integrity through conditional testing.

---

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
Copy the GitHub link and paste it into your terminal:

```bash
git clone [https://github.com/Juanjosegiraldo/Module3_Week_1_2_3_4_Riwi.git](https://github.com/Juanjosegiraldo/Module3_Week_1_2_3_4_Riwi.git)
```

Note: The repository has four folders. Make sure you are in the user_story/week2 folder.

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Launch Development Server:**
   ```bash
   npm run dev
   ````

4. **View Results:** Open your browser at http://localhost:5173 and press F12 (or Cmd+Option+I) to view the logic execution in the Console.
