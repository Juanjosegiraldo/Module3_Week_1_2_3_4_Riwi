# Full Stack Web Development - Module 3 (Riwi)

This repository contains all the projects and exercises developed during the **Module 3: JavaScript Intensive** at Riwi. The focus of this module was building dynamic web applications using modern JavaScript, DOM manipulation, storage, and API integration.

## Repository Structure

The projects are organized by week, each focusing on specific learning milestones:

* **[Week 1](./week1/)**: Introduction to JavaScript Fundamentals & Basic DOM.
* **[Week 2](./week2/)**: Advanced DOM Manipulation & Event Handling.
* **[Week 3](./week3/)**: Data Persistence with Local Storage.
* **[Week 4](./week4/)**: Full Integration (Fetch API, CRUD operations & JSON Server).

    ```bash
    cd user_story/weekX  # Replace X with the week number
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Launch the development server:**
    ```bash
    npm run dev
    ```

*Note: For Week 4 projects, ensure you run `npm run server` in a separate terminal to start the JSON Server.*
